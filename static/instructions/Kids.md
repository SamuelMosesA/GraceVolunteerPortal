# **Grace Kids Instructions**

**Schedule:** Set up @10:40

## **What needs to be done before the kids arrive on Sunday?**

### **Preschool**

* Set up the room, put out the toys and the playmat.  
* Get the snacks, and cups and fill up the water jug.  
* We can use the plastic cups from the SWDA kitchen.  
* Make sure you have the registration cards to register any new kids.

### **Middle Kids**

* Set up the room.  
* Get the snacks, water jug and cups. We can use the plastic cups from the SWDA kitchen.  
* Make sure you have the registration cards to register any new kids.

### **Big Kids**

* Set up the tables and chair in the lobby.  
* Get the Bibles, snacks, water jug and cups. We can use the plastic cups from the SWDA kitchen.  
* Make sure you have the registration cards to register any new kids.

## **What needs to be done after the kids leave on Sundays?**

### **Preschool**

* Pack up the room. If tables were moved, put them back to how you found them.  
* Take the supplies used back to the storage. Please take care to store the bins where they belong, the shelves are labelled.  
* Check if there are enough snacks for the following Sunday. If not, let Diana or Icaro know.  
* Have one person take the cups to the SWDA kitchen and set them on the dishwasher tray.  
* The SWDA staff usually put away the cups, but check if they need help.  
* Message Diana or Icaro with the names and number of kids via WhatsApp.

### **Middle Kids**

* Pack up the room. If tables were moved, put them back to how you found them.  
* Take the supplies used back to the storage. Please take care to store the bins where they belong, the shelves are labelled.  
* Check if there are enough snacks for the following Sunday. If not, let Diana or Icaro know.  
* Have one person take the cups to the SWDA kitchen and set them on the dishwasher tray.  
* The SWDA staff usually put away the cups, but check if they need help.  
* Message Diana or Icaro with the names and number of kids via WhatsApp.

### **Big Kids**

* Move the tables and chairs back to their original position in the lobby.  
* Take the supplies used back to the storage. Please take care to store the bins where they belong, the shelves are labelled.  
* Check if there are enough snacks for the following Sunday. If not, let Diana or Icaro know.  
* Have one person take the cups to the SWDA kitchen and set them on the dishwasher tray.  
* The SWDA staff usually put away the cups, but check if they need help.  
* Message Diana or Icaro with the names and number of kids via WhatsApp.
